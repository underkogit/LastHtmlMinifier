
 


## HtmlMinifier

**HtmlMinifier** — это инструмент для минимизации HTML-кода, который уменьшает его размер, удаляя ненужные пробелы, комментарии и другие избыточные элементы. Проект написан для платформы ESP32.

### Установка

Для установки HtmlMinifier выполните следующую команду:

```bash
cd HtmlMinifier\\HtmlMinifier\\bin\\Debug\\net9.0\\Scripts
```

```bash
npm install html-minifier --save
```

npm install terser
